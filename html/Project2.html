08/01/2021 - 11/01/2021

Introduction

Experienced in full-stack web app programming. I used Vue as the front-end framework, node.js as the back-end, and MongoDB as the database. The goal is to create a web application like Instagram, including features like following a friend, sharing with friends, and more.

Progress

1. Build web application in HTML, CSS and Javascript

2. Build web application in Vue

3. Build Backend in Node.js

4. Create a MongoDB and connect it to the web application.

5. Host it on heroku.

Conclusion

Learned Vue, Bulma, Git, configuration files, asynchronous programming, node.js and mongoDB.
